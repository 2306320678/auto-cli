# auto-cli

![Npm版本](https://img.shields.io/badge/auto-cli_v0.0.1-green)

# 😊简介

- 用于快速搭建前端项目的命令行工具

# 模板

- 快速生成 Vue3 + Ts + Vite + ElementPlus + Pinia + Axios + Mock 的模版

# 安装

```bash
npm install auto-cli -g
```

## 使用

```bash
# 使用模版创建新项目
auto-cli create

# 执行项目文件夹名称并选择模版创建新项目
auto-cli create [project-name]

# 查看脚手架版本
auto-cli --version
auto-cli -v


```
